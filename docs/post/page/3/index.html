Articulo: PAGINAS ESTATICAS

El articulo y la imagen fueron tomados del siguiente link: https://www.webandmacros.com/webestatica.htm Autor del articulo Anonimo OMAR CRUZ Una página estática es un sitio de Internet que muestra el mismo contenido para todos los usuarios, en vez de proporcionar contenido personalizado a la medida de cada usuario, y que no se actualiza a menudo. Las páginas web estáticas, que algunas veces se conocen como páginas planas, contrastan con las páginas web dinámicas que se actualizan frecuentemente y proporcionan información personalizada para usuarios individuales.
